# Contribution Guidelines

Thank you for considering contributing to Awesome AI for Chemistry! This document outlines the guidelines for contributing to this repository.

## How to Contribute

1. **Fork the repository** and create a new branch for your contribution
2. **Add your contribution** following the format below
3. **Submit a pull request** with a clear description of what you're adding

## What to Contribute

We welcome contributions including:
- Research papers on AI applications in chemistry
- Open-source tools and libraries
- Datasets and benchmarks
- Tutorials and educational resources
- Blog posts and community resources

## Guidelines for Adding Content

### Papers

When adding papers, please follow this format:

```markdown
- **Paper Title** - Authors, Conference/Journal (Year) [[Paper]](link)
```

Example:
```markdown
- **SchNet: A Continuous-filter Convolutional Neural Network for Modeling Quantum Interactions** - Sch√ºtt et al., NeurIPS (2017) [[Paper]](https://papers.nips.cc/paper/2017/hash/303ed4c69846ab36c2904d3ba8573050-Abstract.html)
```

### Tools and Libraries

When adding tools, please follow this format:

```markdown
- **Tool Name** - Brief description [[GitHub]](link) or [[Website]](link)
```

Example:
```markdown
- **RDKit** - Open-source cheminformatics toolkit [[GitHub]](https://github.com/rdkit/rdkit)
```

### Datasets

When adding datasets, please follow this format:

```markdown
- **Dataset Name** - Brief description [[Link]](url)
```

## Quality Standards

Please ensure your contributions meet the following criteria:

- **Relevance**: Content must be related to AI applications in chemistry or chemical sciences
- **Quality**: Papers should be from reputable venues (top-tier conferences, peer-reviewed journals)
- **Working Links**: All links must be valid and accessible
- **Proper Formatting**: Follow the existing format in the README
- **No Duplicates**: Check that your contribution isn't already listed
- **English Language**: All descriptions should be in English
- **Alphabetical Order**: When possible, maintain alphabetical order within sections

## Computer Science Focus

This repository focuses on AI for chemistry from a computer science perspective. Preferred content includes:
- Machine learning and deep learning methods
- Novel algorithms and architectures
- Computational approaches
- Software tools and frameworks

## What Not to Include

Please avoid:
- Promotional or commercial content
- Papers without significant AI/ML components
- Broken or paywalled links (when free alternatives exist)
- Content that is not related to chemistry or chemical sciences
- Duplicate entries

## Submitting Pull Requests

When submitting a pull request:

1. **Title**: Use a clear, descriptive title
   - Good: "Add GraphAF paper to Molecular Generation section"
   - Bad: "Update README"

2. **Description**: Briefly explain what you're adding and why it's relevant

3. **One Topic Per PR**: Submit separate pull requests for different types of contributions

4. **Check Your Links**: Verify all links work before submitting

## Questions?

If you have questions about contributing, feel free to:
- Open an issue for discussion
- Ask in the pull request comments

Thank you for helping make this resource better for the community!
